# Task4_Data_Storytelling_and_Hypothesis_Testing
### Objective
To combine dashboard insights with statistical testing to validate regional performance differences.

### Statistical Test Performed
Two-Sample T-Test (Unequal Variances)

### Hypothesis
H0: No significant difference in average sales between West and East regions  
H1: Significant difference in average sales between West and East regions  

### Result
P-value (Two-tail): 0.3469

Since p-value > 0.05, the null hypothesis was not rejected.

### Conclusion
Although West region shows higher total revenue, the difference in average transaction value between West and East is not statistically significant.  
Revenue difference is likely driven by order volume rather than higher per-transaction sales.
